# Dzikir Harian

Web app sederhana untuk membaca Dzikir Pagi dan Dzikir Petang.

## Fitur

- Deteksi otomatis sesi dzikir berdasarkan waktu perangkat.
- Pilihan manual Dzikir Pagi dan Dzikir Petang.
- Tampilan satu dzikir per halaman dengan tombol Sebelumnya dan Berikutnya.
- Lafadz Arab, latin, terjemahan, manfaat, sumber, dan target bacaan.
- Counter bacaan dan checkbox selesai.
- Progress harian, jumlah dzikir selesai, riwayat, dan streak.
- Pengaturan font Arab, font latin, dan ukuran teks.
- Penyimpanan lokal menggunakan Local Storage.
- Manifest dan service worker untuk kebutuhan PWA.
